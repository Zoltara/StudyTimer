This is a Pomodoro based Study Timer webapp:
Features:
* Study Timer with Accountability Partners
* Adjustable Pomodoro timer with focus sessions
* Chat - Add friends
* When a user in the chat quits the pomodoro, it sends an automatic message to the chat, "[name] is out."
* When a user is on a 3 min break, an automatic message is sent - "[name] on a break".
* The user needs to click "Back from Break" button, or an automatic message will be sent to the chat ("[name] lost in break".
* Leaderboard of study streaks.
* Integration with calendar for exam countdowns.
* Messeging panel with notifications.
* All user's timers are synced with the group creator timer.
* Only group creator can set the synced timer and delete the group.
* User can choose their own timer which is not synced with the rest of the group.
* Any user can create a study group.
* Users login with email and password, then join a group with their user name via QR code, code, join a group from the list.
* The app remembres previous sessions.
* This app can run on mobile phone via a browser: https://study-timer-beryl-psi.vercel.app/
